# 🖼️ Image Color Manipulation using Python

## 📌 Overview

This project demonstrates basic **Digital Image Processing** using Python. It loads an image, converts it into a NumPy array, modifies the RGB color channels, and displays the transformed image using Matplotlib.

The project is designed for beginners who want to understand how images are represented as arrays and how pixel-level operations can be used to create different visual effects.

---

## 🚀 Features

- Load images using Pillow (PIL)
- Convert images into NumPy arrays
- Manipulate RGB color channels
- Adjust image brightness and colors
- Display processed images using Matplotlib
- Beginner-friendly implementation

---

## 🛠️ Technologies Used

- Python 3
- NumPy
- Pillow (PIL)
- Matplotlib

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/Image-Color-Manipulation.git
```

Navigate to the project folder:

```bash
cd Image-Color-Manipulation
```

Install the required libraries:

```bash
pip install numpy pillow matplotlib
```

---

## ▶️ Usage

Run the Python script:

```bash
python practice1.py
```

or

```bash
python practice2.py
```

The program will:
1. Load the input image.
2. Convert it into a NumPy array.
3. Modify the RGB values.
4. Display the processed image.
