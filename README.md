Features :
-Load images using Pillow (PIL)
-Convert images into NumPy arrays
-Manipulate RGB color channels
-Apply brightness and color adjustments
-Display original and processed images using Matplotlib
-Beginner-friendly and easy to understand
Technologies Used :
-Python
-NumPy
-Pillow (PIL)
-Matplotlib
Learning Outcomes :
-Understand image representation as NumPy arrays
-Perform basic image enhancement techniques
-Learn RGB channel manipulation
-Practice pixel-level image processing in Python
